Problema 2

Desarrollar el modelo de simulación para el siguiente programa lineal:

Encontrar el máximo de la siguiente función:

Z = 2X1 + 3X2 - X3

s.a.:

X1 + X2 >= 2

1 <= X3 <= 2

100 >= X2 >= 0 (Entero)

0 <= X1 <= 10

Diccionario de Variables

Nº Nombre Sigla Clasificación Unidades Descripción
1 Número Máximo de Iteraciones NMAX Exógena Iteraciones Número máximo de simulaciones
2 Contador C Endógena Iteraciones Variable autoincremental que aumenta en 1 unidad por iteración
3 Variable de Decisión 1 X1 Endógena Unidades Variable de decisión
4 Variable de Decisión 2 X2 Endógena Unidades Variable de decisión
5 Variable de Decisión 3 X3 Endógena Unidades Variable de decisión
6 Función Objetivo Z Endógena Unidades Variable a maximizar
7 Variable de Comparación 1 XC1 Estado Unidades/Iteración Variable de decisión comparativa
8 Variable de Comparación 2 XC2 Estado Unidades/Iteración Variable de decisión comparativa
9 Variable de Comparación 3 XC3 Estado Unidades/Iteración Variable de decisión comparativa
10 Función Objetivo de Comparación ZC Estado Unidades/Iteración Variable a maximizar comparativa
11 Número Aleatorio para XC1 rXC1 Estado [0-1]/Iteración Número pseudo-aleatorio entre 0 y 1
12 Número Aleatorio para XC2 rXC2 Estado [0-1]/Iteración Número pseudo-aleatorio entre 0 y 1
13 Número Aleatorio para XC3 rXC3 Estado [0-1]/Iteración Número pseudo-aleatorio entre 0 y 1

Especificación de Relaciones Funcionales

XC1 = a + (b - a) = 0 + (10 - 0)rXC1
XC2 = ROUND(0 + (100 - 0)rXC2)
XC3 = 1 + (2 - 1)rXC3
ZC = 2XC1 + 3XC2 - XC3
IF ZC > Z THEN Z = ZC X1 = XC1 X2 = XC2 X3 = XC3 ELSE Z = Z X1 = X1 X2 = X2 X3 = X3
